Modifier 5

Index between 0 and 65536, depending on the game mode.

For Capture The Flag, this modifier targets Single Flag mode; the value is the amount of time in ticks (30 per second) before the single flag switches between teams.

For Oddball, this modifier changes the Speed With Ball. 0 = Slow, 1 = Normal, 2 = Fast

For Race, this modifier changes the Team Scoring method. 0 = Min, 1 = Max, 2 = Sum
